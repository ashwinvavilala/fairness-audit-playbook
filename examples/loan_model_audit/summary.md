\# Final Audit Summary — Loan Approval Model

\## 1. Key Findings

\- Significant disparities for Black women over 40.

\- Equal Opportunity not satisfied.

\- Approval rate disparities indicate structural inequities.

\- Calibration issues present.

\## 2. Root Causes

\- Underrepresentation

\- Biased repayment labels

\- Proxy features

\- Feature sensitivity

\## 3. Mitigations Applied

\- Oversampling

\- Reweighting

\- Removing ZIP code

\- Threshold adjustments

\- Fairness-constrained optimization

\## 4. Post-Mitigation Results

\- TPR disparity: 0.19 → 0.07

\- Approval disparity: 0.23 → 0.10

\- Calibration improved

\- Model performance stable
